วิธีคำนวณทีละขั้นตอน
1. Magic Number = 256 − Subnet Mask ของ octet ที่น่าสนใจ
2. Network Address = IP ปัดลงให้เป็น multiple ของ Magic Number ใน octet นั้น
3. Broadcast Address = Network Address + Magic Number − 1
4. First Usable IP = Network Address + 1
5. Last Usable IP = Broadcast Address − 1
6. จำนวน Host ใช้งานได้ = 2^(32−prefix) − 2
ตัวอย่าง: 192.168.10.45 /27
Mask = 255.255.255.224 → Magic Number = 256 − 224 = 32
octet สุดท้ายของ IP = 45 → 45 ÷ 32 = 1 เศษ 13 → Network octet = 1 × 32 = 32
Network = 192.168.10.32
Broadcast = 192.168.10.63 (32+32−1)
First = 192.168.10.33 / Last = 192.168.10.62
Hosts = 2^5 − 2 = 30
